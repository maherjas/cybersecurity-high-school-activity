# Cybersecurity Activity for High School

## About This Activity
An interactive web-based activity teaching password security and social engineering awareness for grades 10-12.

## How to Use
1. Click the link below to open the activity
2. Complete both sections:
   - Password Security (learn to create strong passwords)
   - Social Engineering (learn to recognize online scams)
3. Earn a certificate at the end!

## Live Demo
[Click here to try the activity](https://maherjas.github.io/cybersecurity-high-school-activity/)

## For Teachers
- **Time Required:** 30-45 minutes
- **Materials:** Computer with internet access
- **Learning Objectives:**
  - Students will identify weak vs. strong passwords
  - Students will recognize social engineering tactics
  - Students will apply security best practices

## Topics Covered
- Password strength and security
- Password reuse risks
- Social engineering scams
- Online verification safety
- Two-factor authentication (2FA)

## Technical Details
- Single HTML file - no installation needed
- Works on any modern browser
- No data collected or stored
- Completely free to use and share
